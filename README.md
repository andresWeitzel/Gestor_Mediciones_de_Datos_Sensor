# Gestor Mediciones de Datos Sensor.

**Se desarrolla una GUI en Java con Maven para la gestión de Mediciones de datos  de un Sensor implementando la comunicación Serial con Arduino y aplicando JDBC en MYSQL.**
 
 </br>
